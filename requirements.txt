# 🛡️ Explainable AI for Cyber Threat Detection

> A unified ML framework for detecting Network Intrusions and Phishing attacks,
> with SHAP-based explainability for transparent, trustworthy predictions.

---

## 📌 Project Overview

Cybersecurity systems powered by Machine Learning often act as "black boxes" —
they detect threats but can't explain *why*. This project solves that problem
by combining an ensemble ML model with SHAP (SHapley Additive exPlanations)
to make every prediction interpretable.

**Threats Detected:**
- 🔴 Network Intrusions (using CICIDS 2017 dataset)
- 🟠 Phishing Websites (using PhiUSIIL dataset)

---

## 💡 Novelty

- ✅ Dual-threat detection in a single unified pipeline
- ✅ SHAP explainability on ensemble model (RF + XGBoost)
- ✅ Feature-level explanation for every prediction
- ✅ Comparison against 4 baseline models
